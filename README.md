## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## .env

```bash
$ DATABASE_URL="file:./dev.db"
```

## prisma

```bash
$ npx prisma migrate dev
```

## swagger
Rode a aplicação e acesse o endpoint ```/api``` 
```bash
$ localhost:3000/api
```
